# 🧮 簡易電卓アプリ

5日間のインターン期間中に制作した、HTML / CSS / JavaScript のみで動作する簡易電卓アプリです。  
四則演算をはじめ、バックスペース・符号切り替えなどの機能を実装し、GitHub Pages で公開しています。

🔗 **Demo**：https://nekomikandayo.github.io/calculator-app/  
💻 **Source Code**：https://github.com/nekomikandayo/calculator-app

---

## 📌 概要

JavaScriptの基礎理解を目的として、インターンの課題として個人開発しました。  
単に動くものを作るだけでなく、**境界値分析を実施し、異常系も含めたテスト設計**を行うことで、  
想定外の入力に対してもエラーが発生しない堅牢な設計を意識しています。

---

## 🛠 使用技術

| 技術 | 用途 |
|------|------|
| HTML | ページ構造・ボタンレイアウト |
| CSS | スタイリング・デザイン |
| JavaScript | 演算ロジック・DOM操作・イベント処理 |
| GitHub Pages | 公開・デプロイ |

---

## ✅ 実装機能

| 機能 | 説明 |
|------|------|
| 四則演算 | ＋・−・×・÷ に対応 |
| 小数点入力 | 小数の計算に対応 |
| クリア（C） | 現在の入力のみ消去 |
| 全消去（CE） | 計算全体をリセット |
| バックスペース | 1文字ずつ削除 |
| 符号切り替え（+/-） | 正負の切り替え |

---

## 🎯 工夫した点

- **ファイルを分割**（HTML / CSS / JS）し、保守しやすい構造にした
- UIをシンプルで直感的なデザインにし、誰でも迷わず使えるよう配慮した
- 画像アイコン（バックスペースボタン）を使用して視認性を向上させた
- 相対パスを理解し、GitHub Pages で画像が正しく表示されるよう修正した
- **境界値分析**を実施し、異常系（ゼロ除算・連続演算子入力など）も考慮したテスト設計を行った

---

## 💡 学んだこと

- DOM操作の基本（getElementById・textContent・classList など）
- イベントリスナーの使い方（addEventListener）
- 相対パスと絶対パスの違い
- GitHub Pages での静的サイト公開方法
- キャッシュ問題への対処方法（URLにクエリパラメータを付与するなど）
